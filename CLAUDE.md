# CLAUDE.md

## 项目概述

WorkQuery — 职场技能速查刷题微信小程序。uni-app (Vue 3) + 微信云开发（云函数 + 云数据库），免登录、轻量化。

## 技术栈

- 前端：uni-app 3.x (Vue 3 Composition API) + Pinia + uni-ui
- 后端：微信云函数 (Node.js 18) + 云数据库 (NoSQL)
- 构建：HBuilderX 或 Vue CLI
