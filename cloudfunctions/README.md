# 云函数目录

本目录存放微信云开发的云函数。每个子目录为一个独立云函数。

## 已规划云函数

- `getBanks` — 获取题库列表
- `getQuestions` — 获取题目列表（分页）
- `getKnowledgeCategories` — 获取速查分类
- `getKnowledgeItems` — 获取知识点列表
- `getKnowledgeDetail` — 获取知识点详情
- `search` — 全局搜索
- `syncWrongQuestions` — 同步错题数据
- `syncFavorites` — 同步收藏数据
- `syncProgress` — 同步刷题进度

## 部署

在微信开发者工具中，右键每个云函数目录 → "上传并部署：云端安装依赖"
